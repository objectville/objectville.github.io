Android Studio 查看应用布局

发表于 2018-08-13

Android Studio 有一个可以查看手机应用布局的工具，有一天它不见了

Hierarchy Viewer

以前都是叫这个名字，然后在去年被废弃了。下面这个是替代品

Layout Inspector

随着 Android Studio 的不断更新，这个工具的位置在不断变换 ，直到最近我想起用它时，找了好久才找到。因为在 Android Studio 3 以后，它被移到了 Tools 菜单下，定睛就可以看到了。

然后以为可以愉快的玩耍了，但是窗口是这样的

如果不是自己开发的应用，没法看布局？你以前可不是这样的。

问了度娘，找了谷哥，都是告诉你怎么使用看布局云云，没一个说为啥看不到不是 debug 的应用

就在心灰意冷之际，又定睛看到了一个关键

就是这个小勾，这样才是真正可以愉快玩耍的时候了

年纪大了，眼神果然不好使了，此文毫无技术含量，仅以此记录，万一哪天有个跟我一样眼神不好使的基友没发现这个小地方呢。

最后以一个天猫首页布局结尾
