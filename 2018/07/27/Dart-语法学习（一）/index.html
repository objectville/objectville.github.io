Dart 语法学习（一）

发表于 2018-07-27

这是一篇学习笔记

  • 首先，Dart 和众多语言有相似之处， 可以转译为 JavaScript
  • Dart 中定义的所有变量都是对象，不管是数字，方法还是null都是对象，所有的对象都继承自Object类
  • Dart 是强类型的，但是可以不指定类型，Dart 可以自行推断类型，此时可以用 dynamic修饰
  • Dart 没有 Java 中的 public、protected 和 private，如果要表示私有方法，用_开头命名方法即可
