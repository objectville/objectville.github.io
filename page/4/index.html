Kotlin 中的 val 和 var

发表于 2018-05-16

定义

首先这两个是声明变量的关键字

val（value）—— 不可变引用。使用 val 声明的变量不能在初始化之后再次赋值。它对应的是 Java 的 final 变量

var（variable）—— 可变引用。这种变量的值可以被改变。它对应的是普通（非 final）的 Java 变量
